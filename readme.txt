Приложение содержит 3 раздела: Курс валют, Операции обмена, Отчет.


Раздел "Курс валют" позволяет:
  -просматривать историю курсов валют;
  -вводить курсы валют на текущий и последующие дни.
Если курсы валют за прошедшие дни не были установлены, внести их задним числом уже нельзя.
Установленные значения курсов валют нельзя изменить.


Раздел "Операции обмена" позволяет:
  -вносить операции покупки валюты;
  -вносить операции продажи валюты;
Внесение обменной операции подразумевает собой: выбор вида операции (покупки или продажи), выбор валюты, внесение суммы операции, внесение данных о покупателе/поставщике (серия и номер паспорта, фамилия, имя, отчество).
Все операции создаютя только за текущую дату.
Если на текущую дату курсы валют не установлены - осуществление операций невозможно, о чем и сообщается пользователю при попытке перехода в раздел "Операции обмена".
Если корреспондент уже покупал/продавал валюту (если его паспортные данные зафиксированы в БД), то ФИО такого корреспондента будут выведены из БД в форму при введении его серии-номера паспорта.


Раздел "Отчеты" содержит в себе следующую информацию:
  - Дата операции обмена;
  - Номер операции;
  - ФИО покупателя/поставщика;
  - Сумма прихода (для операций продажи валюты);
  - Сумма расхода (для операций покупки валюты);
  - Итоги по приходным и расходным операциям;
  - Сумма прибыли/убытка за период.
Период отчета определяется пользователем: выбирается дата начала и дата конца периода.
